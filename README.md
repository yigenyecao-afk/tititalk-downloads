# TiTiTalk Downloads

公开下载镜像 — 只放 DMG / EXE 等发布产物，源码仍在私仓。

主入口：[https://tititalk.com/](https://tititalk.com/)

## 加速下载（国内）

把 GitHub 链接前缀加上 `https://mirror.ghproxy.com/`：
- 例：`https://mirror.ghproxy.com/https://github.com/yigenyecao-afk/tititalk-downloads/releases/download/vX.Y.Z/TiTiTalk-X.Y.Z.dmg`
